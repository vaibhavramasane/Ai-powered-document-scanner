📌 Scanner Pro – Web-Based QR & Barcode Scanner

Scanner Pro is a lightweight and user-friendly web application that allows users to scan QR codes and barcodes directly from their browser. Built using modern web technologies, this project demonstrates real-time camera access, efficient data processing, and an intuitive UI.

🚀 Features=>
📷 Real-time QR & barcode scanning using device camera
⚡ Fast and accurate detection
💻 Fully browser-based (no installation required)
📱 Responsive design (works on mobile & desktop)
🔍 Displays scanned results instantly

🛠️ Tech Stack=>
HTML5 – Structure
CSS3 – Styling & responsiveness
JavaScript – Functionality & camera handling
Web APIs – Camera access (MediaDevices API)

🎯 Purpose:
This project was built to demonstrate practical implementation of JavaScript Web APIs and to create a simple yet powerful tool for scanning codes without needing external apps.
